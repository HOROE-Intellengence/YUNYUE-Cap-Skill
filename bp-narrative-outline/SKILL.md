# BP叙事大纲生成器（Step 2 of 3）

工作流位置：头脑风暴锁叙事（Step1）→ **本skill出大纲（Step2）**→ premium-slide-prompt 出美工提示词（Step3）。

## 世界观：BP不是介绍材料，是攻心武器

以下是方法论内核，不可违背。具体页数、顺序、模块划分都可以根据项目灵活变形，甚至可以刻意打破——只要你能说清为什么。

### 1. 一句话锚点是宪法

整个BP必须能压缩成一句核心叙事。每一页出大纲前先问：**这页是否推进这句话的可信度？** 不推进，砍掉，无论内容多好看。旧BP最常见的死法就是"什么都想讲"，每页都对，合起来没有灵魂。

**如果用户尚未锁定核心叙事一句话，停下，先反问共创，不要直接出大纲。** 问法参考："这个项目如果只能让投资人记住一句话，是哪句？" 锁死之前产出的任何大纲都是废纸。

### 2. 反共识是最强开局

最好的叙事不是"我们做得好"，是"整个行业都想错了，只有我们看对了"。寻找反共识的路径：
- 行业默认的成本项，能不能变成收入项？（负成本结构）
- 所有人在抢的位置，是不是真正值钱的位置？（赛道下面的基建位）
- 大家公认的壁垒，是不是真壁垒？（客户关系≠数据壁垒）

反共识必须能被商业结构支撑，喊口号的反共识是自杀。

### 3. 让投资人自己得出结论

- **估值想象力只给类比锚点，严禁做算术。** 给业务形态锚（像谁的商业模式）、时代位置锚（像谁的生态位）、资产属性锚（像什么资产）。你算数字=给投资人砍价的靶子；你给锚点=他自己脑补出百亿结论。他说的，比你说的有力十倍。
- **竞对可以点名，用表格对比，让差距自己说话。**
- **用竞对的话给自己背书**（如引用对手创始人的窗口期判断制造紧迫感）。

### 4. 数字纪律

- 真实数字显性化放大，是最强弹药。
- 没有的数字留 `[ 待补充 ]`，**禁止编造**。留空的表格也要占足视觉分量。
- 数字锚点服务叙事：选"越滚越大"的数（累计数据小时、覆盖门店），慎用"会被追问"的数（利润率、单位经济，除非漂亮）。

### 5. 一页一焦点

每页只允许一个记忆锚点。想强调两件事=拆两页或砍一件。巨大数字、挑衅性断言、大metaphor、狠对比——四选一，不叠加。

## 消费与现金流类项目的FA叙事思考

科技类项目的叙事武器——"技术代差""800亿美元大市场""反共识纠偏"——在消费和现金流类项目里基本失效，因为投资人第一反应就是"这不就是个卖XX的吗"。

但有一条原则跨赛道通用：**最好的叙事不是"我们做得好"，是"整个行业都想错了，只有我们看对了"。** 消费类项目同样需要反共识，只不过反的不是技术路线，而是行业对这门生意的结构性认知。

### 一、先诊断：这类项目为什么"听着一般"

投资人面对消费/现金流类项目时的审美疲劳，通常来自四个开关：

| 疲劳源 | 投资人内心OS | 对应科技类项目的"解药"（消费类用不了） |
|---|---|---|
| 模式一眼看穿 | "不就是开店卖东西吗" | 技术黑盒 -> 制造神秘感 |
| 壁垒不明显 | "别人也能开，凭什么是你" | 专利/数据飞轮 -> 降维论证 |
| 天花板可见 | "开到几千家就到头了" | TAM万亿市场 -> 想象空间 |
| 增长靠堆店 | "增长=烧钱开店，不是真增长" | 技术外溢/多场景扩展 -> 第二曲线 |

所以FA的核心任务不是"把一般讲成不一般"，而是找到一个被整个行业忽略的结构性变量。这个变量一旦被揭示，投资人会重新评估这门生意的价值。

### 二、消费/现金流类项目的三条反共识路径

#### 路径1：负成本结构

行业默认的成本项，能不能变成收入项？

这是消费类项目最有杀伤力的反共识，因为消费生意的成本结构是投资人最容易拆解的。

| 项目类型 | 行业默认成本项 | 反共识重构 |
|---|---|---|
| 外带酒馆连锁 | 门店租金+人力是最大成本 | 自有品牌酒水毛利70%+，门店本质是"品鉴体验+品牌触点"而非利润中心，真正的利润在供应链 |
| 自助售卖机投放 | 机器折旧+点位租金+补货人力 | 机器是数据采集终端，补货路线数据可反哺选品/点位定价模型，数据资产可二次变现 |
| 零食零售 | 采购成本+门店运营 | 自有品牌占比从0到40%时，毛利结构从"搬箱子"变成"品牌商"，单店模型被重构 |
| 情趣用品外卖 | 库存+平台抽佣 | 隐私溢价让客单价远高于电商，30分钟即时性创造的"冲动消费窗口"是电商无法触达的增量市场 |

**FA实操要点：** 不要在BP里喊"我们有供应链优势"，要直接展示一张"传统模型 vs 我们的模型"的单店利润对比表，让投资人自己看到成本结构被翻转了。

#### 路径2：赛道下面的基建位

所有人抢的位置，是不是真正值钱的位置？

消费类项目最容易犯的错误是"跟赛道龙头正面竞争"。反共识的打法是不抢位置，抢位置下面的地基。

- **外带酒馆：** 所有人都在抢"年轻人的酒"这个位置。但真正值钱的是"夜间即时消费场景的入口"。谁控制了这个入口，谁就能叠加夜宵、社交、内容、甚至即时零售。类比：瑞幸不是在做咖啡，是在做"写字楼日间消费入口"。
- **自助售卖机网络：** 所有人都在看"单机产出"。但真正值钱的是"点位网络"。5000台机器覆盖的5000个社区/工厂/医院，本质上是"最后100米的分布式零售基础设施"。这个网络一旦建成，可以叠加任何品类（药品、日用品、鲜食）。机器不是生意，网络才是。
- **情趣用品外卖：** 所有人都在看"情趣用品赛道"。但真正值钱的是"隐私消费即时配送"这个基础设施。30分钟达+隐私包装+无需与店员交互，这套履约能力可以迁移到药品（男科/妇科）、个护（私密品类）等更多隐私敏感型消费。

**FA实操要点：** 在BP的"市场空间"页，不要只画一个TAM/SAM/SOM漏斗。要画一张"表层赛道 -> 底层基建"的分层图，让投资人看到你站的不是赛道，是赛道下面的地基。

#### 路径3：重新定义壁垒

大家公认的壁垒，是不是真壁垒？

消费类项目里最常见的质疑是护城河。这里的变形原则是：**品牌/门店/供应链关系/点位资源本身都不是壁垒，复制成本才是壁垒。**

| 大家以为的壁垒 | 为什么不是真壁垒 | 真正的壁垒是什么 |
|---|---|---|
| 品牌知名度 | 品牌是结果不是壁垒，而且烧钱就能砸 | 品牌背后的用户复购行为数据+基于数据的选品/定价迭代闭环 |
| 门店数量 | 门店多不等于壁垒深，关店也快 | 加盟商盈利模型的可复制性。不是合同绑定，是"加盟商跟着你能赚钱"的经济绑定 |
| 供应链关系 | 供应商关系随时可以被替代 | 自有品牌占比+SKU效率模型。别人能抄你的店，抄不了你的供应链效率 |
| 点位资源 | 点位可以买，也可以被抢 | 点位网络的密度效应。同一区域50台机器的补货成本远低于分散的50台，密度本身就是壁垒 |

**FA实操要点：** 竞争优势页不写"我们有什么"，写"别人要复制我们，需要付出什么代价"。用一张"复制成本矩阵"让投资人自己算出"这东西不容易抄"。

### 三、分领域的具体思考与预案

#### 1. 外带酒馆连锁/加盟

**反共识钩子：** 行业默认酒馆=堂食社交空间。但Z世代的酒类消费正在从"坐下来喝"变成"带走喝"。外带酒馆不是"缩小版的酒吧"，是"酒类消费的DTC渠道革命"。

**投资人最关心的核心问题：**

- 加盟商能不能持续赚钱？不是头部门店赚钱，是中位数门店赚钱。
- 自有品牌酒水的占比能到多少？这是毛利翻转的关键。
- 夜间消费场景的延展性如何？

**叙事变形：**

- 封面钩子不要写"新一代年轻人酒馆"，写"中国有300万夜店，没有一家解决'带走喝'这个问题"。
- 飞轮逻辑：门店扩张 -> 自有品牌渗透 -> 供应链规模效应 -> 单店毛利提升 -> 加盟商吸引力增强 -> 更快扩张。
- 估值锚点：业务形态锚=瑞幸的渠道逻辑（不是星巴克的空间逻辑）；时代位置锚=茶颜悦色在长沙验证的"区域密度即品牌"逻辑。

**必须准备的预案：**

- 酒类监管风险（未成年人保护、广告限制） -> 准备合规说明页。
- "加盟模式=割韭菜"的质疑 -> 准备加盟商存活率数据+单店回本周期分布图。不要只给平均值，要给中位数和P25/P75。
- 同店增长 vs 新店增长的拆分 -> 如果同店增长为负，诚实面对，解释原因和改善措施。

#### 2. 情趣用品外卖店

**反共识钩子：** 行业默认情趣用品=电商生意。但"即时性+隐私性"是电商无法满足的双重需求。30分钟达+无标识包装+无人交互，解决的是"冲动且不愿等待"的增量需求，不是存量转移。

**投资人最关心的核心问题：**

- 平台依赖度（美团/饿了么流量控制权）是最大风险。
- 客单价和复购率能否支撑单店模型。
- 品类延展性：能不能从情趣用品扩展到更多隐私消费品类。

**叙事变形：**

- 不要把项目定位为"情趣用品零售商"，定位为"隐私消费即时零售基础设施"。
- 竞争地图：不是跟电商比价格，跟电商比"即时性+隐私性"。这是两个不同的需求场景。
- 增长弹药：展示分层客户结构。情趣用品是入口品类，药品（男科/妇科）、个护是延展品类。

**必须准备的预案：**

- 政策敏感性 -> 正面回应合规框架，不回避。
- 平台抽佣风险 -> 展示自有私域流量占比（微信群/小程序复购），证明不完全依赖平台。
- 社会观感问题 -> 准备"投资人沟通话术"，帮助FA在向LP转述时有足够的框架支撑。

#### 3. 无人售货/自助售卖机

**反共识钩子：** 行业默认无人售货=低效零售补充。但在特定场景（夜间社区、工厂园区、医院），无人售货不是"补充"，是"唯一可行的即时供给方案"。有人店在这些场景的经济模型根本跑不通。

**投资人最关心的核心问题：**

- 单机产出够不够？日均销售额 vs 机器折旧+点位租金。
- 点位获取能力：点位是稀缺资源，怎么锁定？
- 密度效应何时显现？区域内机器数量达到多少时，补货成本/管理成本开始下降？

**叙事变形：**

- 不要按"单机经济模型"讲故事，按"网络效应"讲。前100台是验证，1000台是网络雏形，5000台是基础设施。
- 飞轮：点位密度上升 -> 补货效率提升 -> 单机运营成本下降 -> 可触达场景增加 -> 点位密度继续上升。
- 估值锚点：不是零售公司，是"线下分布式零售节点网络"。类比丰巢在快递柜领域的网络价值。

**必须准备的预案：**

- "机器是重资产，折旧压力大" -> 准备机器生命周期内的总产出 vs 总成本对比，以及二手残值说明。
- "点位被抢怎么办" -> 展示已签排他协议/长期租约的占比。
- "品类受限" -> 展示不同场景的品类组合实验数据，证明可扩展性。

#### 4. 餐饮连锁

**反共识钩子：** 餐饮赛道最拥挤，但品类生命周期是有规律的。反共识的核心是判断某个品类正处于生命周期上升期的哪个位置。不是"我们又一家火锅店"，是"砂锅品类正处于麻辣烫2018年的位置"。

**投资人最关心的核心问题：**

- 同店增长趋势：开店增速 vs 同店增速的剪刀差。
- 供应链标准化程度：能不能做到"千店一味"。
- 加盟管控体系：食品安全风险如何系统化控制。

**叙事变形：**

- 封面钩子：用品类对标制造时间差感，例如"砂锅赛道今天就像2018年的麻辣烫，跑出千亿品牌的窗口期还有18个月"。
- 市场空间：不画TAM，画"品类生命周期曲线"。标注当前位置，用已成熟品类的演进路径做类比锚。
- 证明页：用同店增长率（而非总门店数）作为核心记忆锚点。

**必须准备的预案：**

- 食品安全事故预案 -> 展示食品安全管理体系，不是口号，是具体流程和追溯系统。
- 品类生命周期衰退 -> 展示第二品类的储备/测试进展。
- 加盟商管控 -> 展示数字化巡检系统+加盟商淘汰机制。

#### 5. 快消品/零食零售/美妆/珠宝

这几个领域各有特点，但共同的反共识路径是：**渠道变革创造的结构性机会。**

| 领域 | 结构性变量 | 反共识切入点 | 叙事核心 |
|---|---|---|---|
| 快消品 | 抖音电商/社区团购重塑渠道 | 不是"又一个新品牌"，是"渠道红利期的品类占位者"。先在渠道验证单SKU效率，再反哺品牌 | 单SKU的渠道效率数据（ROI、复购率） |
| 零食零售 | 自有品牌重构毛利结构 | 不是"卖零食的店"，是"零食供应链效率平台" | 自有品牌占比 x 毛利改善曲线 |
| 美妆 | 成分党+供应链成熟 | 不是"营销驱动品牌"，是"研发驱动的成分占位者"。在特定成分赛道建立心智壁垒 | 成分专利/独家供应链+复购率 |
| 珠宝 | 黄金按克销售+工费模式 | 不是"传统珠宝店"，是"黄金消费场景的渠道下沉者"。三四线城市的黄金消费升级 | 单店投资回收期+加盟商年化ROI |

### 四、叙事框架的变形原则

消费/现金流类项目需要对默认叙事骨架做如下变形：

| 默认原则 | 科技类的用法 | 消费/现金流类的变形 |
|---|---|---|
| 反共识开局 | 技术路线之争："整个行业的技术方向错了" | 结构性变量之争："整个行业对这门生意的成本结构/渠道逻辑/品类周期判断错了" |
| 角色化痛点 | 技术痛点链：客户A -> 技术B -> 场景C | 消费者+加盟商+供应商三角：三个角色的需求如何被一个结构同时接住 |
| 数字纪律 | "越滚越大"的累计数据 | 单店经济模型是命根子。单店投资额、回本周期、日均流水、坪效、人效，必须真实且经得起追问 |
| 估值锚点 | 时代位置锚/资产属性锚 | 渠道效率锚、品类周期锚、现金流结构锚 |
| 飞轮 | 数据-算法-场景多回路 | 门店-数据-供应链-品牌-门店多回路，关键是每一步都有可量化的验证指标 |
| 一页一焦点 | 巨大数字/挑衅性断言 | 狠对比（传统模型 vs 我们的模型）/真实数字（回本周期/复购率） |

### 五、FA必须准备的三套预案

#### 预案1：项目确实天花板有限

不是所有消费项目都能讲成百亿故事。FA需要诚实评估：

- 如果天花板就是区域龙头 -> 叙事重心放在"区域密度即壁垒"，退出路径指向被行业龙头并购。
- 如果天花板是几千家店但增速放缓 -> 叙事重心放在"从渠道公司进化为品牌/供应链公司"，用第二曲线撑估值。
- 如果确实就是个小生意 -> 建议创始人不要硬融资，或者接受较低估值找产业资本。

#### 预案2：投资人"不性感"的质疑

准备"现金流即壁垒"的叙事。在当前资本环境下，能自我造血的项目反而更有韧性：

- "这个项目不需要烧钱验证需求，需求已经存在，我们验证的是效率模型。"
- "上一轮消费品泡沫里，拿了大钱的项目反而被钱烧坏了组织；这个项目靠自身现金流滚动，组织韧性更强。"
- 核心记忆锚点：如果单店3-6个月回本，加盟商年化ROI超30%，这就是最强投资逻辑，不需要任何包装。

#### 预案3：政策与合规风险

适用于酒类/情趣/无人售货等品类：

- 在BP中设专门的合规页面，正面回应，不回避。
- 准备"投资人沟通话术包"，帮助FA在向LP/IC转述时有足够的框架支撑。
- 对于情趣用品等敏感赛道，考虑在BP封面和前3页完全不提品类，先讲"隐私消费即时零售"的框架，再揭晓具体品类。利用"钩子 -> 共鸣 -> 揭晓"的节奏制造认知反转。

### 六、一句话总结

消费和现金流类项目的FA叙事核心 = **平凡生意中的不平凡结构**。

FA的工作不是给烂生意化妆，而是帮创始人找到那一个被行业忽略的结构性变量。如果找到了，就把它放大成反共识钩子；如果找不到，就诚实告诉创始人这可能是门好生意但不是个融资故事，建议靠自身现金流滚动或者接受现实估值。

比把项目讲好更重要的，是知道什么项目该用什么框架讲，什么项目不该硬讲。

## 默认叙事骨架：反共识型

这是默认牌组，可整体使用、可抽换、可增删。节奏内核是：**钩子 → 共鸣 → 揭晓 → 论证 → 证明 → 想象 → 行动**

| 段落 | 页面功能 | 说明 |
|---|---|---|
| 钩子 | 封面 | 副标题即钩子，埋一个"这怎么可能"的疑问 |
| 共鸣 | 角色化痛点 | 2-4个具体主体+各自需求，看似无关实为一条链。**不讲抽象概念，讲人** |
| 揭晓 | 我们的位置 | 一个结构同时接住所有需求 + 行业地位一句话 |
| 论证 | 反共识 | 行业假设错在哪，点名对比 |
| 论证 | 结构解剖 | 讲清商业机制为什么成立（客户为什么答应） |
| 论证 | 飞轮 | 多回路自增强，拒绝单环PPT飞轮 |
| 证明 | 落地进展 | 硬数字，配"这只是开头"的心智暗示 |
| 证明 | 增长弹药 | 分层客户结构，越往下基数越大 |
| 证明 | 竞争地图 | "我们不在赛道里"式的降维定位 |
| 想象 | 估值锚点 | 三重类比，零数字 |
| 行动 | 战略节奏 | 用外部时间压力（窗口期）倒逼节奏感 |
| 行动 | 团队 | 排序服务叙事（商业叙事则CEO打头） |
| 行动 | 融资 | 资金用途的逻辑要与核心叙事一致 |
| 收口 | 封底 | 与封面首尾呼应，slogan三连 |

角色化痛点开场是签名打法，默认保留；其余页面按项目自由取舍。

## 每页输出格式（对接 premium-slide-prompt）

### P{N} · {页面功能}
- **核心信息**：这页投资人必须记住的一件事（给step3的第一问直接递答案）
- **标题**：中文定稿文案，直接上片，一字不改
- **模块1**：{容器类型} | 内容（≤60字）
- **模块2/3**：同上，每页≤3模块
- **记忆锚点**：本页唯一焦点（数字/断言/metaphor/对比，四选一）
- **底注**：可选
- **待补数据**：`[ 留空标记 ]`
- **排版建议**：一两句，给step3方向不锁死细节

## 交付规则

- 大纲交付后附：**整 deck 节奏总览** + **待补数据清单** + **核心叙事一句话**
- 用户可能多次抽卡：重新生成时**换叙事切角**（换反共识点、换metaphor体系、换开场角色组合），不是换措辞。措辞级微调不算抽卡。
- 交付时主动指出：哪几页是全deck的胜负手，哪几页可以将就。

## 常见死法（避开）

- 没锁叙事就动手 → 产出精美的废纸
- 每页都想讲三件事 → 每页都没人记住
- 用形容词代替结构（"行业领先""不可复制"） → 投资人自动免疫
- 自己做估值算术 → 递刀给人砍价
- 技术自嗨页（CTO视角的架构图） → 稀释商业叙事的可信度
- 抄行业通用话术（"三曲线增长""生态赋能"） → 泯然众BP
